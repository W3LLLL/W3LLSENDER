> How To Install ? 

1) __`sudo apt install -y git`_
2) _`git clone https://github.com/W3LLLL/W3LLSENDER.git`_
3) _`cd W3LLSENDER`_
4) _`chmod 777 install.sh`_
5) _`bash install.sh`_

> All this command should paste on your terminal / putty software
