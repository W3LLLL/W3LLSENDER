> How To Install ? 

1) `sudo apt install -y git`
2) `git clone https://github.com/W3LLLL/W3LLSENDER.git`
3) `cd W3LLSENDER`
4) `chmod 777 install.sh`
5) `bash install.sh`

> All this command should paste on your terminal / putty software
